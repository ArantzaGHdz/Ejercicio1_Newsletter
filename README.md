## 📝 Reflexión

Aprendí como crear la estructura de una pagina web en HTML y su diseño con CSS. Descubrí como implementar formularios donde el usuario puede
introducir datos (en este caso una dirección de correo electrónico) y aplicar un diseño estructurado con margenes, tamaños, padding, colores, fuentes tipograficas y estilos de sombreado.

Tuve algunos problemas al empezar el diseño debido a que no recordaba como centrar los elementos del diseño, pero investigue en Internet. Desconocía como hacer sombras difuminadas y colocar el texto indicativo (descubrí que se llama placeholder) dentro del input del email, igualmente encontre la respuesta en Internet como foros de programación.
